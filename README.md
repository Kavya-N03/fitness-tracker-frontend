# Fitness Tracker Frontend

A responsive **Fitness Tracker web application** built using **React**, integrated with a Django REST API backend.  
The application helps users set fitness goals, log daily activities and meals, and track calorie intake, calories burned, and overall daily progress.

---

**Features**

- **User Authentication**
  - Secure user login and registration
  - JWT-based authentication flow

- **Goal Management**
  - Set personal fitness goals
  - View progress towards set goals

- **Activity Tracking**
  - Log daily activities such as walking, running, swimming, yoga, etc.
  - View calories burned from activities

- **Meal & Nutrition Tracking**
  - Log meals consumed by the user
  - Track nutritional values including calories, protein, fats, and carbohydrates

- **Daily Progress Dashboard**
  - View daily summary of:
    - Calories consumed
    - Calories burned
    - Protein, fats, and carbohydrates intake
  - Track fitness progress by date

- **Responsive UI**
  - Optimized for mobile and desktop devices
